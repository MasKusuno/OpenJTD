# Ichitaro OpenOffice Filter Research

The historical OpenOffice Ichitaro Document Filter is now a first-priority research artifact for rjtd.

The downloaded `.oxt` is stored locally under `third-party/ichitaro-filter/`.

Tracked findings are recorded in [openjtd-spec/rfc/0002-ichitaro-openoffice-filter.md](../openjtd-spec/rfc/0002-ichitaro-openoffice-filter.md).

## Working Rule

Use this artifact only as a compatibility and naming reference unless legal review says otherwise.

The included license restricts decompilation and reverse engineering, so rjtd development must remain clean-room:

- metadata is useful;
- OpenOffice filter/type registration is useful;
- package file tree is useful;
- visible strings can guide hypotheses;
- implementation code must not be copied or reconstructed from the DLL.
